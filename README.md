# 耶稣模拟器

一个纯 HTML、CSS 与 JavaScript 制作的单页互动体验。

## 本地打开

直接打开 `site/index.html` 即可预览。为获得和线上一致的效果，也可以使用任意静态网页服务器打开 `site` 文件夹。

## 部署到 GitHub Pages

项目已经包含自动部署配置：

1. 将整个项目推送到 GitHub 仓库的 `main` 分支。
2. 打开仓库的 **Settings → Pages**。
3. 在 **Build and deployment → Source** 中选择 **GitHub Actions**。
4. 等待名为 **Deploy to GitHub Pages** 的任务完成，即可访问网站。

网页成品位于 `site` 文件夹，不依赖数据库、账号或第三方服务。
